# 2026CounterApp

A simple, elegant counter application with increment, decrement, and reset functionality.

## Features

- **Clean, modern UI** with gradient backgrounds and smooth animations
- **Responsive design** that works on mobile and desktop
- **Visual feedback** with color-coded numbers:
  - Green for positive numbers
  - Red for negative numbers  
  - Gray for zero
- **Smooth hover effects** and transitions
- **Single HTML file** - no external dependencies

## How to Use

1. Click the **"+ Increment"** button to increase the counter
2. Click the **"- Decrement"** button to decrease the counter
3. Click the **"Reset Counter"** button to set the counter back to zero
